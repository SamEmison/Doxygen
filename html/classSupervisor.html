Detailed Description

Represents a supervisor, which is a type of employee that supervises other employees.

The Supervisor class inherits from Employee and adds an attribute for the number of employees that the supervisor manages (numSupervised). It overrides methods such as print to include supervisor-specific details and calculatePay to compute the pay based on the supervisor's hourly rate and the number of hours worked.
