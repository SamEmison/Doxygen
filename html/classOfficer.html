Detailed Description

Represents an officer, which is a type of employee with additional "evilness" attribute.

The Officer class inherits from Employee and extends it with an evilness attribute. It overrides methods like print to display the officer's details and calculatePay to compute the pay based on hourly rate and the evilness level.
