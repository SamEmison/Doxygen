\doxysubsection{Detailed Description}
Represents a supervisor, which is a type of employee that supervises other employees. 

The {\ttfamily \mbox{\hyperlink{classSupervisor}{Supervisor}}} class inherits from {\ttfamily \mbox{\hyperlink{classEmployee}{Employee}}} and adds an attribute for the number of employees that the supervisor manages ({\ttfamily num\+Supervised}). It overrides methods such as {\ttfamily print} to include supervisor-\/specific details and {\ttfamily calculate\+Pay} to compute the pay based on the supervisor\textquotesingle{}s hourly rate and the number of hours worked.
