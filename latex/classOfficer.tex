\doxysubsection{Detailed Description}
Represents an officer, which is a type of employee with additional \char`\"{}evilness\char`\"{} attribute. 

The {\ttfamily \mbox{\hyperlink{classOfficer}{Officer}}} class inherits from {\ttfamily \mbox{\hyperlink{classEmployee}{Employee}}} and extends it with an {\ttfamily evilness} attribute. It overrides methods like {\ttfamily print} to display the officer\textquotesingle{}s details and {\ttfamily calculate\+Pay} to compute the pay based on hourly rate and the {\ttfamily evilness} level.
